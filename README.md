# Banking System SQL Analysis

This project simulates a banking database system and demonstrates SQL data analysis skills.

## Database Tables
- Customers
- Accounts
- Transactions
- Loans
- Branches

## Project Files
- create_tables.sql – database schema
- insert_data.sql – sample data
- analysis_queries.sql – business analysis queries

## Business Questions Answered
- Top 3 customers with the highest balances
- Customers with multiple active loans
- Fraudulent transactions detection
- Total loan amount issued per branch
- Suspicious large transactions
- Transactions from different countries in short time

## Technologies Used
- SQL Server
- Relational Database Design
- Data Analysis
